# Proxy Quiz Funnel

Простой MVP лендинга + квиза под search-intent трафик по прокси-запросам.

## Стек
- HTML
- CSS
- JavaScript

Без React, без тяжёлых фреймворков, без лишней сложности.

## Что делаем
Делаем короткую веб-воронку:

1. Входной hero-экран
2. Короткий quiz на несколько шагов
3. Экран результата с подходящими офферами
4. Клики в офферы
5. Простая аналитика по основным событиям

## Основные принципы
- Максимально простая структура
- Быстрая загрузка
- Удобная ручная правка
- Чистый адаптив без переусложнения
- Mobile-first подход

## Структура проекта
index.html          # первый экран / hero
quiz.html           # квиз
result.html         # результат / офферы

css/
  style.css         # общие стили
  responsive.css    # адаптив

js/
  main.js           # логика hero
  quiz.js           # логика квиза
  result.js         # логика результата
  analytics.js      # события аналитики
  offers.js         # офферы / данные / маппинг
  ## Что важно в разработке
- Не тащить лишние библиотеки
- Не усложнять архитектуру
- Не делать длинные анимации
- Не ломать скорость загрузки
- Все тексты должны легко правиться руками
- Все основные параметры офферов должны храниться просто и понятно

## Адаптив
Нужно корректно отображать:
- desktop
- mobile

На мобилке:
- текст сверху
- CTA сразу виден
- визуал ниже
- ничего не расползается

## Аналитика
Минимально отслеживаем:
- view_entry_hero
- click_cta_pick_proxy
- quiz_start
- quiz_complete
- result_view
- offer_click

## Цель MVP
Быстро собрать и протестировать рабочую прокладку без лишней технической сложности.
